# El Grito Mexicano

A beautiful website for an authentic Mexican restaurant featuring handcrafted dishes made from traditional recipes.

## Features

- Modern, responsive design
- Showcase of popular Mexican dishes
- Gallery of restaurant images
- Easy-to-navigate menu

## Getting Started

This is a static HTML/CSS website. Simply open `index.html` in your browser.

## Deployment

This site is deployed via GitHub Pages. 

## Technologies Used

- HTML5
- CSS3
- Responsive Design
